# Gamblify

Displaying sports winners in a modern capacity using cutting edge predictive AI backed by statistics.


## Features

- Multiple Sports
- Multiple Leagues
- Updated Stats
- Cross platform




## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
